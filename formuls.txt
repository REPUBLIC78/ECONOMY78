TC  - total cost             общие издержки +
FC  - fixed cost             постоянные издержки +
VC  - variable cost         переменные издержки +
Q   - quantity               колличество
P   - price                  цена

AFC - Average fixed costs    cредние постоянные затраты
AVC - Average variable costs cредние переменные затраты
ATC - Average total costs    средние общие затраты (себестоимость)

TR  - total revenue          общая выручка
AR  - average revenue        средняя выручка на товар

MC  - margin costs           дополнительные издержки
MR  - margin revenue         дополнительная выручка

Pr  - profit                 прибыль
Qcr - critical quantity      критическое количество


TC = VC + FC 
TC = AFC * Q + AVC * Q
TC = TR - ((P - ATC) * Q)
TC = TR - Pr
TC = ATC * Q

FC = TC - VC
FC = (AFC * Q + AVC * Q) - VC
FC = AFC * Q
FC = Qcr * (P - AVC)

VC = TC - FC
VC = AVC * Q
VC = (AFC * Q + AVC * Q) - FC

ATC = AFC + AVC
ATC = TC / Q
AFC = FC / Q
AVC = VC / Q

TR = P * Q
AR = TR / Q

MC = TC(Q) - TC(Q - 1)
MC = (TC)'
MR = TR(Q) - TR(Q - 1)

Pr = TR - TC
Pr = (P - ATC) * Q

Qcr = FC/(P - AVC)
